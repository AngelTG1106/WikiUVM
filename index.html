CSS ''LA REVOLUCIÓN''  

CON LA LLEGADA DE LA ERA DE LA WEB, LOS DESARROLLADORES SE ENFOCABAN EN CREAR SITIOS WEB ATRACTIVOS. SIN EMBARGO TODO EL CONTENIDO: PALABRAS, IMAGENES Y BOTONES SE ALINEABAN RÍGIDAMENTE SIN CONTROL SOBRE EL COLOR, LA TIPOGRAFÍA O EL ESPACIO ENTRE ELLOS. POR LO QUE AL INICIO DEL MUNDO DIGITAL, CONSEGUIR UN DIEÑO ATRACTIVO PARA UNA PÁGINA WEB LUCIERA SOLIA SER UN RETO. EL HTML ESTABA LIMITADO A ESTRUCTURAR EL CONTENIDO, PERO NO TENÍA PODER SOBRE SU APARIENCIA. SIN EMBARGO, SURGIO UNA NUEVA HERRAMIENTA: CSS, LLEGÓ COMO UN SALVADOR DEL DISEÑO. NO REEMPLAZABA A HTML, SOLO LO COMPLEMETARIA. EL HTML SEGUÍA SIENDO EL ESQUELETO QUE ORGANIZABA LOS ELEMENTOS, PERO AHORA CSS, PODÍA DARLE ESTILO, COLOR Y VIDA A LA ESTRUCTURA. ERA COMO SI LA WEB HUBIERA APRENDIDO A VESTIRSE.

CSS ESTA EN CONSTANTE EVOLUCION, AÑADIENDO PROPIEDADES COMO TRANSICIONES, ANIMACIONES, ETC CON LOS QUE SE PUEDEN CREAR SITIOS WEB INNOVADORES PERMITIENDO SITIOS MÁS DINÁMICOS, INTUITIVOA Y ATRACTIVOS QUE CON EL PASO DEL TIEMPO VA SUPERNADO SUS PROPIOS LIMITES.
